# Metabolomic Signatures of Kidney Dysfunction

Computational metabolomics project analyzing renal tissue LC-MS datasets to identify metabolic signatures associated with ischemic and inflammatory stress in kidney transplantation.

## Project Overview

This project investigates whether metabolite profiles can distinguish renal tissue samples exposed to different perfusion conditions and provide insight into biological pathways associated with transplant injury and dysfunction.

Using publicly available metabolomics datasets, data preprocessing, statistical analysis, and dimensionality reduction techniques were applied to characterize metabolic variation between experimental groups.

## Methods

* Data cleaning and preprocessing using Pandas and NumPy
* Log transformation and missing value imputation
* Feature standardization
* Principal Component Analysis (PCA)
* Statistical comparison of metabolite abundance patterns
* Biological interpretation of metabolomic signatures

## Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* SciPy
* Matplotlib
* Seaborn

## Key Findings

* Identified clear metabolic separation between perfusion conditions.
* PCA captured the majority of variation across samples, demonstrating distinct metabolic profiles.
* Results highlighted metabolite patterns associated with ischemic and inflammatory stress relevant to kidney transplantation research.
